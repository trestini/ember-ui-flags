ember-ui-flags
==============================================================================

Simple addon for displaying world flags in emberjs applications. CSS work is
based on https://github.com/wobblecode/flat-flags and flags design is based on
https://dribbble.com/shots/1211759-Free-195-Flat-Flags.

Installation
------------------------------------------------------------------------------

```
ember install ember-ui-flags
```
